Boost Build System V2 tracker

All bugs, planned features and enhancements for Boost.Build V2 are kept in a online tracker, awailable at http://zigzag.cs.msu.su:7814/

For read only access, please do the following:

  1. Point your browser to the above URL.
  2. Type "anonymous" as username and password.
  3. To see all issues, go to "Select query" menu, and choose "Active"
  4. To see issues planned for the next milestone, choose "Next Milestone".

If you want to create new issues and add comments, you'd need read-write access. To obtain it, please do the following:

  1. Point your browser to the above URL.
  2. Click on the "create an account" link and follow the instructions.
  3. Once you've registered and logged in, click "request roles" link and request "Observe" role in "Boost->Build" module.
