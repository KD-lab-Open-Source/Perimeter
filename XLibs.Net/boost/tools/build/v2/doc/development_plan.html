Boost Build Development Plan

  • Milestone 2 (October 25, 2002)

    Boost buildable with gcc and also usable as part of another project.

  • Milestone 3 (May 21, 2003)

    Two gcc versions and two other compilers are supported. Some other tools (e.g. lex and BoostBook) are implemented.

  • Milestone 4 (June 18, 2003)

    More tools implemented.

  • Milestone 5 (Jule 2, 2003)

    Improvements and bugfixes with dependency scanning. Support for i18n tools. Testing framework for Boost.

  • Milestone 6 (Jule 22, 2003)

    Bugfixes.

  • Milestone 7 (Aug 1, 2003)

    Optimizations.

  • Milestone 8 (TBD)

    Notion of relevant properties. Bugfixes.

  • Milestone 9 (TBD)

    Improve gcc toolset. Automatically compute right libraries order.

  • Alpha (TBD)

    Optimization. Using buildable tools: consider using bison, which is itself build with Boost.Build.

  • Beta (TBD)

    Feature-complete milestone.

  • 2.0 Release (TBD)

    Bugfixes
